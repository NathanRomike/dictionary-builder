# Dictionary Builder - Java

##### This application is to test my knowledge of java. 1.15.2016.

#### By **Nathan Romike**

## Description

> This application exhibits my comprehension of creating java applications with object oriented development. Review my work and tell me what you think. Lets make awesome stuff together!

## Setup

>View this application deployed to [Heroku](https://dictionary-builder.herokuapp.com/).

**Or**

>Clone repository to local machine, compile and run the program in terminal by entering `$ gradle run`, and navigate to the [localhost](http://localhost:4567/).

## Technologies Used

* HTML
* CSS
* Bootstrap
* Java
* Velocity
* Spark
* FluentLenium
* Heroku
